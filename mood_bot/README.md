# Mood Tracker Bot

Телеграм-бот для трекинга настроения и самочувствия. Каждый пользователь ведёт свой личный дневник — данные изолированы по Telegram ID.

---

## Как записывать

Пишешь сообщение в свободной форме, оценка может быть в любом месте текста:

```
проснулась с мыслями о нём, +5
-8 ссора с мужем
хороший завтрак +3
+10 путешествие с мужем
```

**Шкала оценки: от -10 до +10**

---

## Указание времени

Запись сохранится с указанным временем, а не временем отправки сообщения:

```
11:00 завтракал с женой +3
в 5 вечера прогулка +7
11 утра хороший кофе +4
утром в 11 позавтракал +3
в 3 ночи не мог уснуть -5
```

Поддерживаемые форматы периода суток: `утра / утром`, `дня / днём`, `вечера / вечером`, `ночи / ночью`

---

## Промежутки времени

```
11-15 работал +3
с 11 до 15 работал +3
9:30-18:00 офис +2
```

В данных время отображается как диапазон: `11:00–15:00`

---

## Команды

| Команда | Описание |
|---|---|
| `/stats` | Статистика за сегодня |
| `/stats 7` | За последние 7 дней |
| `/stats 01.04.2025 16.04.2025` | За конкретный период |
| `/export` | Excel-файл за сегодня |
| `/export 7` | Excel за последние 7 дней |
| `/export 01.04.2025 16.04.2025` | Excel за период |
| `/delete` | Удалить последнюю запись |
| `/deleteall` | Удалить все свои записи |
| `/help` | Справка |

---

## Вывод статистики

Каждая строка в `/stats` выглядит так:

```
* 2026-04-16, 11:00–15:00, работал 🟢 "+3"
* 2026-04-16, 17:00, прогулка 🟢 "+7"
* 2026-04-16, 23:00, не мог уснуть 🔴 "-5"
```

Цвет оценки: 🟢 положительная · 🔴 отрицательная · 🔵 ноль

## Структура проекта

```
mood_bot/
├── bot.py          — точка входа, обработчики команд
├── db.py           — SQLite: инициализация, запись, выборка
├── parser.py       — парсинг оценки и времени из текста
├── excel.py        — генерация .xlsx через openpyxl
├── requirements.txt
├── Dockerfile
└── fly.toml        — конфигурация деплоя на Fly.io
```

---

## Деплой

Бот запущен на [Fly.io](https://fly.io) в регионе Amsterdam.  
База данных SQLite хранится на постоянном томе `/data/mood.db`.

Обновление:

```bash
fly deploy
```
